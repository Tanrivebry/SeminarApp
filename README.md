# 📱 SeminarApp

Aplikasi Android sederhana berbasis **Kotlin** untuk kebutuhan registrasi seminar. Project ini dibuat sebagai bagian dari pembelajaran pengembangan aplikasi Android.

---

## 🚀 Fitur

* 📝 Formulir pendaftaran seminar
* 👤 Input data pengguna (nama, email, dll)
* 🎨 Tampilan UI sederhana dan bersih
* ⚡ Aplikasi ringan dan mudah digunakan

---

## 🛠️ Teknologi yang Digunakan

* Kotlin
* Android Studio
* XML Layout
* Gradle

---

## 📂 Struktur Project

```
SeminarApp/
 └── app/
     └── src/
         └── main/
             ├── java/com/example/seminarapp/
             │    └── MainActivity.kt
             ├── res/
             │    ├── layout/
             │    └── values/
             └── AndroidManifest.xml
```

---

## ▶️ Cara Menjalankan Project

1. Clone repository ini

```bash
git clone https://github.com/username/SeminarApp.git
```

2. Buka menggunakan Android Studio
3. Tunggu proses Gradle Sync selesai
4. Klik tombol **Run ▶️** untuk menjalankan aplikasi
